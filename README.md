React version of mine projects portfolio.

## Deployment on Vercel

1. Push this code to a GitHub repository
2. Connect your GitHub account to Vercel
3. Import the repository
4. Vercel deploy  

## Local Development

```bash
npm install
npm start

```
---

Project Structure
/src/components - React components

/src/data - Data files for projects, courses, skills

/src/styles - CSS styles

/src/utils - Utility functions

/public/img - Image assets (copy your images here)

 

### Features
- React 18 with functional components

- Dark/Light theme toggle
- Responsive design
- Project cards with fallback images
- Course table with scroll
- Office skills section
- Tech stack visualization
- Mobile-friendly navigation
